Project Overview

This web application allows gym staff to manage memberships, payment plans, and member check-ins. It was built using C#, Razor Pages, and Entity Framework Core, with full CRUD operations and search/sort functionality.

Key Features

  • Create, update, and delete membership plans
  • Track member info, join dates, and payments
  • Search and sort members by name or join date
  • Responsive UI with Bootstrap styling

Technologies Used

  • C#, ASP.NET Core, Razor Pages
  • Entity Framework Core (EF Core)
  • SQL Server LocalDB
  • Bootstrap 5
  • Visual Studio Code
