Project Overview

This project focused on designing a secure, normalized database system to support operations at BSA Urgent Care. The system manages staff, patients, rooms, appointments, medical records, billing, and more. It was developed with healthcare compliance and data accuracy in mind.

Key Features

  • Fully normalized SQL tables (3NF)
  • Custom business rules and access control for hospital roles
  • Accurate tracking of appointments, treatments, and records
  • Test data and queries included

Technologies Used

  • SQL
  • Entity Relationship Modeling
  • Draw.io for ERD
  • VS Code
