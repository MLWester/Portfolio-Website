Project Overview

This console-based application helps apartment leasing staff efficiently log, search, and manage incoming packages for residents. It allows fast interactions through a menu-driven interface built entirely in C#.

Key Features

  • Log new packages with resident name, apartment number, and notes
  • List all packages with pickup status
  • Search for packages by resident name
  • Mark packages as picked up
  • Console-based, user-friendly interface

Technologies Used

  • C#
  • .NET Console Application
  • Visual Studio Code
